Key Projects

Z-Prime AI - AI-Assisted Primer Design & Genomic Surveillance Platform
In Development
Final-Year Research Project · Copperbelt University · 2026 – Present

Leading the development of a Python-based primer design platform optimised for resource-constrained laboratory environments across Zambia and sub-Saharan Africa. The engine implements nearest-neighbour thermodynamics (SantaLucia 1998 parameters), automated hairpin ΔG and self-dimer detection, GC clamp evaluation, and homopolymer run checking. Features a dual-path AI interpretation layer - Claude API for online journal-standard analysis, locally hosted Ollama LLM for offline field use - ensuring the platform functions even in settings with intermittent internet connectivity. Surveillance modules target Vibrio cholerae, Mycobacterium tuberculosis, and Copperbelt environmental pathogens under a One Health framework. CLI engine complete and validated; Streamlit web dashboard in development.

Malaria Genomic Surveillance Pipeline (v2.1)
Completed
Bioinformatics Development Project · Copperbelt University · 2025 – 2026

Built and validated an end-to-end variant-calling pipeline for detecting K13 artemisinin resistance mutations in Plasmodium falciparum isolates from public ENA datasets. Implements allele-verified clinical annotation (position + ALT nucleotide verification), depth-aware quality filtering (QUAL ≥ 30, DP ≥ 10), and automated resistance reporting against WHO-validated K13 mutation catalogs. Fully Dockerised for cross-platform reproducibility and parameterised via a configurable CLI. Pipeline validated using paired-end Illumina data from the MalariaGEN Pf7 project.
